HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The title element is for metadata purposes and is not shown on the page itself (though it does appear on the tab in-browser
  • lang defines which language the page is in. "en" is for English. You only speak that language, so remember that one
  • Don't forget to add a closer for each element/tag. The program seems to do it automatically, but forgetting could break stuff

CSS

  • Margin = amount of space outside the element
  • Padding = amount of extra space inside the element but outside the content
  • Don't. Forget. The. Freaking. Semicolons.
